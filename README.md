# Python健身房管理系统简介

## 核心功能
- 健身项目管理
- 会员卡管理
- 健身活动管理
- 健身器材管理
- 员工管理
- 会员管理
- 个人中心
- 教练信息管理
- 指导项目管理

## 适用场景
- 健身房运营管理
- 健身服务提供
- 会员信息管理
- 活动组织与执行
- 器材维护与调度

## 技术特点
- 使用Django框架构建，保证系统稳定性和安全性
- 基于MySQL数据库，数据存储可靠
- 前端采用Vue技术，提升用户体验
